del (functions in module plumbing )

Description

Low-level call to delete to a specific REST endpoint from FamilySearch

Usage

del(url[, opts][, headers][, responseDataExtender]);

Parameters

  • url – {String} –

    may be relative

  • opts(optional) – {Object=} –

    options to pass to the http function specified during init

  • headers(optional) – {Object=} –

    options headers

  • responseDataExtender(optional) – {Function=} –

    function to extend response data

Returns

{Object} –

a promise that behaves like promises returned by the http function specified during init
