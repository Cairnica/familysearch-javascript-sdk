getCouple (functions in module spouses )

Description

Get information about a couple relationship The response includes the following convenience functions

  • getId() - id of the relationship
  • getHusbandId()
  • getWifeId()
  • getFacts() - array of facts decorated with fact convenience functions as described for getPerson
