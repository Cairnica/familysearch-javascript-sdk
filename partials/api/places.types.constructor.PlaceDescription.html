PlaceDescription
type in module places

Description

Place description returned by the Place Authority.

Usage

PlaceDescription();

Methods

  • $getFullName()

    Returns
    String

    The fully qualified name (e.g. includes the name of parent jursdictions) of this place as listed in the display properties

  • $getJurisdictionDetails()

    Get the full details of the jurisdiction's PlaceDescription. The promise will fail if no jurisdiction is available.

    Returns
    Object

    A promise for the response of retrieving the details for the jursdiction

  • $getJurisdictionSummary()

    Get a summary of the jurisdiction's place description, when available. Useful for place descriptions returned by places.functions:getPlaceDescription if you just want the names and ids of all place descriptions in the jurisdiction chain. If you want more details then use places.types:constructor.PlaceDescription#$getJurisdictionDetails.

    Returns
    PlaceDescription

    A summary PlaceDescription for this PlaceDescription's jurisdiction.

  • $getName()

    Returns
    String

    The name of this place as listed in the display properties.

  • $getType()

    Returns
    String

    The type as listed in the display properties.

  • $setJurisdiction(A)

    Parameters
    ParamTypeDetails
    AObject

    json object representing the new jursidication data, or a PlaceDescription object.

Properties

  • display

    Returns
    Object

    includes name, fullName, and type attributes

  • id

    Returns
    string

    place id

  • identifiers

    Returns
    Object

    map of identifiers to arrays of values

  • lang

    Returns
    string

    The language of the Place Description. See http://www.w3.org/International/articles/language-tags/

  • latitude

    Returns
    number

    Degrees north or south of the Equator (0.0 degrees). Values range from −90.0 degrees (south) to 90.0 degrees (north).

  • longitude

    Returns
    number

    Angular distance in degrees, relative to the Prime Meridian. Values range from −180.0 degrees (west of the Meridian) to 180.0 degrees (east of the Meridian).

  • names

    Returns
    TextValue[]

    An array of names. The preferred value is first.

  • temporalDescription

    Returns
    Object

    A description of the time period to which this place description is relevant.

  • type

    Returns
    string

    A URI used to identify the type of a place.
