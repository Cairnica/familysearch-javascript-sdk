getMultiSourceDescription (functions in module sources )

Description

Get multiple source descriptions at once by requesting them in parallel

FamilySearch API Docs

editable example

Usage

getMultiSourceDescription(sdids[, params][, opts]);

Parameters

  • sdids – {Array} –

    Ids or SourceRefs of the source descriptions to read

  • params(optional) – {Object=} –

    pass to getSourceDescription currently unused

  • opts(optional) – {Object=} –

    pass to the http function specified during init

Returns

{Object} –

promise that is fulfilled when all of the source descriptions have been read, returning a map of source description id to response
