getMultiAgent (functions in module user )

Description

Get multiple agents at once by requesting them in parallel

Usage

Parameters

    Ids of the agents to read

    pass to getAgent currently unused

  • opts(optional) – {Object=} –

    pass to the http function specified during init

Returns

{Object} –

promise that is fulfilled when all of the agents have been read, returning a map of agent id to response
