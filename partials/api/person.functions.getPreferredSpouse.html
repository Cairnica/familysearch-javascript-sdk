getPreferredSpouse
functions in module person

Description

Get the preferred spouse if any for this person and this user. The response includes the following convenience function

  • getCoupleRelationships() - an array of Couple relationships; the array is of length 1 if this user has set a preferred spouse for this person, or 0 otherwise
