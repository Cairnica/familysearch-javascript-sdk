getRelationshipsToChildren (functions in module person )

Description

Get the relationships to a person's children. The response includes the following convenience functions

  • getChildIds() - an array of string ids
  • getRelationships() - an array of ParentChild relationships
  • getPerson(pid) - if the persons parameter has been set, this function will return a Person for a person in the relationship
