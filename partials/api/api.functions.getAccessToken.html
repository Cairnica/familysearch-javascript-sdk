getAccessToken (functions in module api )

Description

Get the access token for the user.

Call this function before making any calls that require authentication. The SDK caches the access token returned so you don't need to; you just need to ensure that the promise that is returned by this function resolves before making calls that require authentication

Usage

getAccessToken([authCode]);

Parameters

  • authCode(optional) – {String=} –

    optional auth code from getAuthCode; if not passed in, this function will call getAuthCode first

Returns

{Object} –

a promise of the access token.
