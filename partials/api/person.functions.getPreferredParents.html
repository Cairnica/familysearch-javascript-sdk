getPreferredParents
functions in module person

Description

Get the preferred parents if any for this person and this user. The response includes the following convenience function

  • getChildAndParentsRelationships() - an array of ChildAndParents relationships; the array is of length 1 if this user has set preferred parents for this person, or 0 otherwise
