getSourceRefsQuery (functions in module sources )

Description

Get the people, couples, and child-and-parents relationships referencing a source The response includes the following convenience functions

  • getPersonIdSourceRefs() - get an array of IdSourceRefs from the response
  • getCoupleIdSourceRefs() - get an array of IdSourceRefs from the response
  • getChildAndParentsIdSourceRefs() - get an array of IdSourceRefs from the response
