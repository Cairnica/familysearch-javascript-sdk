getOrdinancesPolicy
functions in module ordinances

Description

Get the policy that must be agreed to by the user in order to reserve an LDS ordinance. The policy text is retrieved from the response by calling response.getData().

API Docs

Usage

getOrdinancesPolicy([format][, language]);

Parameters

ParamTypeDetails
format
(optional)
String

Response format: text or html (defaults to text)

language
(optional)
String

Value of the Accept-Language header that determines the language of the policy

Returns

Object

promise for the response
