MemoryPersona
type in module memories

Description

Memory Persona (not a true persona; can only contain a name and a media artifact reference)

FamilySearch API Docs

Usage

MemoryPersona(client[, data]);

Parameters

ParamTypeDetails
clientFamilySearch

FamilySearch sdk client

data
(optional)
Object

an object with optional attributes {name, memoryArtifactRef}. To create a new memory persona, you must set memoryArtifactRef and name. name can be a Name object or a fullText string. NOTE memory persona names don't have given or surname parts, only fullText
