Couple
type in module spouses

Description

Couple relationship

Two methods to note below are save and delete. save persists the changes made to husband, wife, and facts; delete removes the relationship.

Usage

Couple(client[, data]);

Parameters

ParamTypeDetails
clientFamilySearch

FamilySearch sdk client

data
(optional)
Object

an object with optional attributes {husband, wife, facts}. husband and wife are Person objects, URLs, or ids. facts is an array of Facts or objects to be passed into the Fact constructor.
