  • save(url, changeMessage)

    Create a new source reference (if this source reference does not have an id) or update the existing source reference

    NOTE: there's no refresh parameter because it's not possible to read individual source references; however, the source reference's id and URL are set when creating a new source reference.

    NOTE: the person/couple/childAndParents id and the source description are not updateable. Only the tags are updateable.
