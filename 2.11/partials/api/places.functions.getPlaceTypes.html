getPlaceTypes
functions in module places

Description

Get a list of all available Place Types.

  • getList() - get the VocabularyList from the response
  • getPlaceTypes() - get an array of the VocabularyElements from the response

API Docs

Usage

getPlaceTypes(lang);

Parameters

ParamTypeDetails
langString

Optional. The language you want the display data returned in, when available. In the browser this defaults to the browser's language. On the server this defaults to 'en'.

Returns

Object

promise for the response
