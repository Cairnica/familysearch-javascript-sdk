getSourceAttachments
functions in module sources

Description

Get the people, couples, and child-and-parents relationships that have the given URL attached as a source. Use getSourceRefsQuery to get attachments for an existing source description.

The response includes the following convenience functions

  • getPersonSourceRefs() - get an array of person SourceRefs from the response
  • getCoupleSourceRefs() - get an array of couple relationship SourceRefs from the response
  • getChildAndParentsSourceRefs() - get an array of child and parent relationship SourceRefs from the response
  • getSourceDescriptions() get an array of SourceDescriptions from the response
  • getSourceDescription(id) get the SourceDescription with the specified source description id from the response

FamilySearch API Docs. This method uses the source parameter of the Source References Query resource.

Usage

getSourceAttachments(url);

Parameters

ParamTypeDetails
urlString

url of the source references query resource of a source description
