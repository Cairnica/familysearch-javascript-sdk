getSourceRefsQuery
functions in module sources

Description

Get the people, couples, and child-and-parents relationships referencing a source description. To get attachments for a URL, use getSourceAttachments The response includes the following convenience functions

  • getPersonSourceRefs() - get an array of person SourceRefs from the response
  • getCoupleSourceRefs() - get an array of couple relationship SourceRefs from the response
  • getChildAndParentsSourceRefs() - get an array of child and parent relationship SourceRefs from the response

FamilySearch API Docs This method uses the description parameter of the Source References Query resource.

Usage

getSourceRefsQuery(url);

Parameters

ParamTypeDetails
urlString

url of the source references query resource of a source description

Returns

Object

promise for the response
