getAccessToken
functions in module authentication

Description

Get the access token for the user. Call this function before making any requests that require authentication.

You don't need to store the access token returned by this function; you just need to ensure that the promise returned by this function resolves before making calls that require authentication.

This method does not allow your site to handle any piece of the auth process. Use getOAuth2AuthorizeURL if you want to initiate and recieve the redirects yourself.
